eit-status(1)
=============
:man source:   eit {eitversion}
:man manual:   eit {eitversion}


NAME
----
eit-status - show repository status


SYNOPSIS
--------
eit status [-h] [<repo>]


INTRODUCTION
------------
Show repository status (such as: *configured mirrors*,
*current branch*, *unstaged packages*, *packages ready for upload*, etc).



OPTIONS
-------
"eit status" supports the following options which alters its behaviour.


POSITIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~~~
*<repo>*::
    repository

OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit
