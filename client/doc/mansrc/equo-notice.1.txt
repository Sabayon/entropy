:man source:   equo {equoversion}

NAME
----
equo-notice - repository notice board reader


SYNOPSIS
--------
equo notice [-h] <repo> [<repo> ...]


INTRODUCTION
------------
Read Repository Notice Board.



OPTIONS
-------
"equo notice" supports the following options which alters its behaviour.


POSITIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~~~
*<repo>*::
    repository

OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit
