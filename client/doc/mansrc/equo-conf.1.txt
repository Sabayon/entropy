============
:man source:   equo {equoversion}
:man manual:   equo {equoversion}


NAME
----
equo-conf - manage package file updates


SYNOPSIS
--------
equo conf [-h] {update} ...


INTRODUCTION
------------
Manage package file updates.



OPTIONS
-------
"equo conf" supports the following options which alters its behaviour.


OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit

ACTION
~~~~~~
*update*::
    update configuration files
