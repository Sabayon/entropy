=============
:man source:   equo {equoversion}

NAME
----
equo-cache - manage Entropy Library Cache


SYNOPSIS
--------
equo cache [-h] {clean} ...


INTRODUCTION
------------
Manage Entropy Library Cache.



OPTIONS
-------
"equo cache" supports the following options which alters its behaviour.


OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit

ACTION
~~~~~~
*clean*::
    clean Entropy Library Cache
